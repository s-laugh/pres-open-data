Options

Centralize Data

Centralizing data, provide single point for exposing data

Benefits

  • Manage Interoperability
  • Single point for access

Risks

  • Consolidating data from legacy
  • Single point of failure
  • Long-tem maintainability (large scale solution)
  • Big Brother Perception

Open APIs

Have the groups that own an operate data, expose it though APIs

Benefits

  • Manage Interoperability
  • Data Ownership stays localized
  • Scaling is simplified

Risks

  • Varying Interoperability
  • Onboarding
